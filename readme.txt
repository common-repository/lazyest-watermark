This plugin adds Watermarks to your Lazyest Gallery Images

== Description ==

Watermark plugin for [Lazyest Gallery](http://wordpress.org/extend/plugins/lazyest-gallery/)

Adds Google Chart Images and Text as watermark to your images

== Installation ==

0. Install activate and configure Lazyest Gallery
1. Install Lazyest Watermark using the WordPress plugin installer
2. Activate
